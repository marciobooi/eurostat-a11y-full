ACCESSIBILITY AUDIT SUITE — FULL CODE REVIEW
=============================================
DESCRIPTION:
This suite provides a comprehensive, automated accessibility (A11y) audit for the Eurostat Sankey 
Data Visualization tool. It combines industry-standard Axe Core scans with custom, manual 
simulations for complex interactions like timeline sliding, modal focus traps, and chart 
navigation to ensure compliance with WCAG 2.1 AA standards.

Last Updated: 2026-03-20 | Status: 9/9 scripts fully optimized

KEY FEATURES & RECENT IMPROVEMENTS
==================================

1. Hierarchical Test Selection (test-selector.html)
   -----------------------------------------------
   [x] Implemented parent/child dependency model.
   [x] Sub-audits (Chart, Legend, Language, Dropdown) require their parent to be selected.
   [x] Deselecting a parent automatically disables/clears its children.
   [x] UI includes "requires Step X" badges and indented sub-cards.

2. Proactive Translation Service (undefined-text-tester.js)
   --------------------------------------------------------
   [x] REFACTORED: Now exports a standalone `performScan` service.
   [x] Integrated `performScan` into every modal/dropdown/tutorial test.
   [x] CATCHES: "undefined" or "null" text even if modals are unmounted when closed.
   [x] COVERAGE: Scans DocumentElement, Iframes (recursive), and 13+ ARIA/Data attributes.
   [x] PATTERN: Switched to non-boundary regex (/undefined/i) to catch partial matches like "year undefined".

3. Scenario-Based Reporting (report-generator.js)
   ----------------------------------------------
   [x] REFACTORED: Primary grouping is now by **Test Scenario** (Step), not rule ID.
   [x] Each step has a collapsible card showing its unique metadata.
   [x] Within each step, rule violations are sorted by impact (Critical > serious).
   [x] Detailed occurrences include full HTML code and "Copy" buttons for debugging.

4. Robust Tester logic
   -------------------
   [x] Navbar: Corrected focus sequence and added aria-hidden container detection.
   [x] Select: Added 5b sub-check for opening heavy dropdown only when selected.
   [x] Floating: Legend audit now part of the 4b selectable sub-flow.
   [x] Node Modal: Improved 3-tier click logic and chart hand-off.
